### 🌍 Поддерживаемые языки

- **🇰🇬 Кыргызский (kg)** - Кыргызча
- **🇷🇺 Русский (ru)** - по умолчанию

### 📄 Переведенные секции

Все секции главной страницы (`app/page.tsx`) переведены на оба языка:

1. **Навигация** (`nav`)
   - Логотип, меню, кнопка входа

2. **Главная секция** (`hero`)
   - Заголовок, описание, кнопки, статистика

3. **Возможности** (`features`)
   - Умный поиск станций
   - Бронирование
   - Быстрый старт

4. **Как это работает** (`howItWorks`)
   - 4 шага процесса зарядки

5. **Оплата** (`payment`)
   - Способы оплаты
   - Пример счета

6. **Приложение** (`app`)
   - Описание приложения
   - Функции
   - Статистика

7. **Тарифы** (`pricing`)
   - AC зарядка
   - DC быстрая зарядка
   - DC ультра зарядка

8. **Призыв к действию** (`cta`)
   - Заголовок и кнопки

9. **Подвал** (`footer`)
   - Ссылки на разделы
   - Контакты
   - Правовая информация

## 🚀 Как использовать

### Быстрый старт (3 шага):

1. **Импортируйте функцию**:
```typescript
import { getTranslations } from '@/app/i18n';
```

2. **Загрузите переводы**:
```typescript
const t = await getTranslations('ru', 'landing');
```

3. **Используйте в компонентах**:
```typescript
<h1>{t.hero.title}</h1>
```

### Полная интеграция:

См. файл `INTEGRATION.md` для подробных инструкций.

## 📝 Следующие шаги

### Для интеграции в проект:

1. **Вариант А: Простая интеграция**
   - Добавьте состояние языка в `app/page.tsx`
   - Используйте `LanguageSwitcher` в навигации
   - Замените хардкод текстов на переводы

2. **Вариант Б: С роутингом (рекомендуется)**
   - Создайте структуру `app/[locale]/`
   - Настройте middleware для редиректов
   - Используйте динамические роуты

### Для расширения:

1. **Добавить новый язык**:
   - Создайте папку `app/i18n/locales/en/`
   - Добавьте конфигурацию в `config.ts`
   - Переведите все JSON файлы

2. **Добавить новую страницу**:
   - Создайте `about.json` для каждого языка
   - Добавьте интерфейс в `types.ts`
   - Обновите `TranslationMap` в `utils.ts`

1. **Автодополнение** - TypeScript подсказывает доступные ключи
2. **Проверка типов** - Ошибки при неправильных ключах
