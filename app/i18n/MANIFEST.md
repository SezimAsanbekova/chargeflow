## 🧪 Тестирование

### Ручное тестирование:
```bash
# Запустить dev сервер
npm run dev

# Открыть в браузере
# http://localhost:3000

# Проверить:
# - Переключение языков
# - Все тексты переведены
# - Нет ошибок в консоли
# - Адаптивность на мобильных
```

### Автоматическое тестирование:
```bash
# Проверка JSON
node -e "JSON.parse(require('fs').readFileSync('app/i18n/locales/kg/landing.json'))"
node -e "JSON.parse(require('fs').readFileSync('app/i18n/locales/ru/landing.json'))"

# Сравнение структуры (требует jq)
diff <(jq -S 'keys' app/i18n/locales/kg/landing.json) \
     <(jq -S 'keys' app/i18n/locales/ru/landing.json)
```
