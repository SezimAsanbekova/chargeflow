# 🎨 Визуальное руководство по локализации

## 🖼️ Как это выглядит

### Переключатель языков

#### Закрытое состояние
```
┌─────────────────────────────────────┐
│  🇰🇬 Кыргызча  ▼                    │
└─────────────────────────────────────┘
```

#### Открытое состояние
```
┌─────────────────────────────────────┐
│  🇰🇬 Кыргызча  ▲                    │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│  🇰🇬 Кыргызча              ✓       │  ← Активный (зеленый фон)
│  🇷🇺 Русский                        │  ← Неактивный (hover эффект)
└─────────────────────────────────────┘
```

## 📊 Сравнение языков

### Длина текстов

| Секция | Русский | Кыргызский | Разница |
|--------|---------|------------|---------|
| Навигация | Короткие | Средние | +20% |
| Заголовки | Средние | Длинные | +30% |
| Описания | Длинные | Очень длинные | +40% |

**Рекомендация:** Используйте `text-ellipsis` для длинных текстов.

## 💡 Советы по дизайну

### 1. Учитывайте длину текста

```typescript
// ❌ Плохо - текст может обрезаться
<button className="w-32">
  {t.hero.downloadApp}
</button>

// ✅ Хорошо - адаптивная ширина
<button className="px-8 py-4 whitespace-nowrap">
  {t.hero.downloadApp}
</button>
```

### 2. Используйте ellipsis для длинных текстов

```typescript
<p className="truncate max-w-xs">
  {t.features.smartSearch.description}
</p>
```

### 3. Тестируйте оба языка

```typescript
// Проверьте, как выглядит на обоих языках
<div className="grid grid-cols-2 gap-4">
  <div>RU: {t.hero.title}</div>
  <div>KG: {t.hero.title}</div>
</div>
```

## 🌈 Цветовая схема

### Переключатель языков

```css
/* Фон */
bg-emerald-500/10      /* Основной фон */
hover:bg-emerald-500/20 /* При наведении */

/* Границы */
border-emerald-500/30   /* Основная граница */
hover:border-emerald-500/50 /* При наведении */

/* Текст */
text-gray-300          /* Неактивный */
text-emerald-400       /* Активный */

/* Dropdown */
bg-[#0f2d26]          /* Фон dropdown */
bg-emerald-500/20     /* Активный элемент */
hover:bg-emerald-500/10 /* Hover элемент */
```

## 📐 Размеры и отступы

```css
/* Кнопка переключателя */
px-4 py-2              /* Padding */
rounded-lg             /* Скругление */
gap-2                  /* Расстояние между элементами */

/* Dropdown */
w-48                   /* Ширина */
mt-2                   /* Отступ сверху */
rounded-lg             /* Скругление */

/* Элементы списка */
px-4 py-3              /* Padding */
gap-3                  /* Расстояние между иконкой и текстом */
```

## 🎬 Анимации

```css
/* Стрелка dropdown */
transition-transform   /* Плавный поворот */
rotate-180            /* Поворот при открытии */

/* Hover эффекты */
transition            /* Плавный переход */
hover:bg-emerald-500/10 /* Изменение фона */

/* Появление dropdown */
opacity-0 → opacity-100
scale-95 → scale-100
```
