# 🗑️ Удаление аккаунта

## ✅ Реализовано

Полноценная функция удаления аккаунта с подтверждением паролем и модальным окном.

---

## 📋 Процесс удаления

```
1. Пользователь нажимает "Удалить аккаунт"
   ↓
2. Открывается модальное окно с предупреждением
   ↓
3. Пользователь вводит пароль для подтверждения
   ↓
4. Система проверяет пароль
   ↓
5. Удаляются ВСЕ данные пользователя
   ↓
6. Автоматический выход из системы
   ↓
7. Редирект на главную страницу
```

---

## 🔒 Безопасность

### Требования:
- ✅ **Авторизация** - только авторизованный пользователь может удалить свой аккаунт
- ✅ **Подтверждение паролем** - обязательный ввод пароля
- ✅ **Проверка пароля** - сравнение с хешем в БД
- ✅ **Предупреждение** - четкое описание последствий

### Что удаляется:
- ✅ **Профиль пользователя** (User)
- ✅ **Баланс** (UserBalance) - CASCADE
- ✅ **Автомобили** (Vehicle) - CASCADE
- ✅ **Бронирования** (Booking) - CASCADE
- ✅ **Сессии зарядки** (ChargingSession) - CASCADE
- ✅ **Платежи** (Payment) - CASCADE
- ✅ **Счета** (Invoice) - CASCADE
- ✅ **Аккаунты OAuth** (Account) - CASCADE
- ✅ **Сессии NextAuth** (Session) - CASCADE

**Благодаря `onDelete: Cascade` в Prisma схеме, все связанные данные удаляются автоматически!**

1. Войдите в систему
2. Откройте http://localhost:3000/profile

7. **Ожидаемый результат:**

1. Откройте модальное окно удаления

1. Откройте модальное окно удаления
2. Нажмите "Отмена" или [X]
3. **Ожидаемый результат:**
   - ✅ Модальное окно закрывается
   - ✅ Аккаунт НЕ удален
   - ✅ Остаетесь на странице профиля

1. Откройте модальное окно удаления

   - ✅ Кнопка "Удалить" неактивна
   - ✅ Нельзя нажать

1. Откройте модальное окно удаления

## 📊 API Endpoint

### POST /api/user/delete-account

Удаляет аккаунт текущего пользователя.

**Request:**
```json
{
  "password": "MySecure123!"
}
```

**Response (Success):**
```json
{
  "success": true,
  "message": "Аккаунт успешно удален"
}
```

**Response (Error - неверный пароль):**
```json
{
  "error": "Неверный пароль"
}
```

**Response (Error - не авторизован):**
```json
{
  "error": "Не авторизован"
}
```

## 🔧 Технические детали

### API (app/api/user/delete-account/route.ts):

```typescript
// 1. Проверка авторизации
const session = await getServerSession(authOptions);

// 2. Проверка пароля
const isPasswordValid = await bcrypt.compare(password, user.passwordHash);

// 3. Удаление пользователя (CASCADE удалит все связанные данные)
await prisma.user.delete({
  where: { id: user.id },
});
```

### Страница профиля (app/profile/page.tsx):

