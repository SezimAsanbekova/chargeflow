## 📦 Что реализовано

### ✅ Backend API (6 endpoints)
- `GET /api/vehicles` - Список всех автомобилей
- `POST /api/vehicles` - Добавление нового автомобиля
- `GET /api/vehicles/[id]` - Получение одного автомобиля
- `PATCH /api/vehicles/[id]` - Обновление автомобиля
- `DELETE /api/vehicles/[id]` - Удаление автомобиля
- `POST /api/vehicles/[id]/set-active` - Активация автомобиля

### ✅ Frontend Pages (3 страницы)
- `/vehicles` - Список автомобилей с управлением
- `/vehicles/add` - Форма добавления нового автомобиля
- `/vehicles/edit/[id]` - Форма редактирования

### ✅ Функции
- Добавление неограниченного количества автомобилей
- Редактирование всех параметров
- Удаление с подтверждением
- Выбор активного автомобиля
- Автоматическое управление активным статусом
- Валидация данных на клиенте и сервере
- Адаптивный дизайн для всех устройств

### ✅ Данные автомобиля
**Обязательные:**
- Марка (brand)
- Модель (model)
- Год выпуска (year)
- Тип коннектора (CCS2, CHAdeMO, Type2, GB/T)
- Максимальная мощность зарядки (кВт)
- Ёмкость батареи (кВт·ч)

**Опциональные:**
- Текущий уровень заряда (0-100%)
- Активный статус

## 📊 Бизнес-логика

### Активный автомобиль:
1. Первый автомобиль автоматически становится активным
2. Только один автомобиль может быть активным
3. При активации нового - предыдущий деактивируется
4. При удалении активного - активируется первый оставшийся

### Валидация:
- Мощность > 0 кВт
- Батарея > 0 кВт·ч
- Заряд: 0-100%
- Год: последние 30 лет
- Коннектор: только из списка

## 🧪 Тестовые сценарии

### Сценарий 1: Первый автомобиль
1. Войдите в аккаунт
2. Откройте "Мои автомобили"
3. Нажмите "Добавить автомобиль"
4. Заполните форму
5. Сохраните
6. ✅ Автомобиль должен стать активным автоматически

### Сценарий 2: Второй автомобиль
1. Добавьте второй автомобиль
2. Не ставьте галочку "Сделать активным"
3. Сохраните
4. ✅ Первый автомобиль остается активным

### Сценарий 3: Переключение активного
1. Откройте список автомобилей
2. Найдите неактивный автомобиль
3. Нажмите "Сделать активным"
4. ✅ Автомобиль становится активным
5. ✅ Предыдущий активный становится неактивным

### Сценарий 4: Редактирование
1. Нажмите иконку карандаша
2. Измените уровень заряда
3. Сохраните
4. ✅ Изменения сохранены

### Сценарий 5: Удаление
1. Нажмите иконку корзины
2. Подтвердите удаление
3. ✅ Автомобиль удален
4. ✅ Если был активным - другой стал активным
