## 💡 Особенности

### Форматирование телефона:
```typescript
// Ввод: 996501979495
// Вывод: +996 501 979 495

const formatPhone = (value: string) => {
  const numbers = value.replace(/\D/g, '');
  if (numbers.startsWith('996')) {
    const formatted = numbers.slice(3);
    return `+996 ${formatted.slice(0, 3)} ${formatted.slice(3, 6)} ${formatted.slice(6, 9)}`;
  }
  return value;
};
```

### Валидация:
- Номер должен начинаться с 996
- Длина: 12 цифр (996 + 9 цифр)
- Формат: +996 XXX XXX XXX
