# 🏠 Редирект с Landing страницы

## ✅ Что реализовано

### Логика редиректа:

```
Пользователь открывает / (главную страницу)
   ↓
Проверка авторизации
   ↓
┌─────────────────────┬─────────────────────┐
│  НЕ авторизован     │   Авторизован       │
│  ↓                  │   ↓                 │
│  Показать Landing   │   Редирект в        │
│  с кнопкой "Вход"   │   /profile          │
└─────────────────────┴─────────────────────┘
```

---

## 📋 Поведение

### Для НЕавторизованных пользователей:
- ✅ Показывается **Landing страница**
- ✅ В навигации кнопка **"Вход"**
- ✅ Нет меню пользователя
- ✅ Доступны все секции: Features, How it Works, Pricing, App

### Для авторизованных пользователей:
- ✅ **Автоматический редирект** в `/profile`
- ✅ Landing страница **не показывается**
- ✅ Показывается "Переход в профиль..." во время редиректа

---

**Для авторизованных:**

**Для авторизованных:**
```
┌─────────────────────────────────────────┐
```

**Для неавторизованных:**
```
Редирект → /auth/signin
```

## 🔄 Процессы

### Процесс 1: Новый пользователь

```
1. Открывает / → Landing страница
2. Нажимает "Вход" → /auth/signin
3. Переключается на "Регистрация"
4. Регистрируется → Код на email
5. Вводит код → Вход в систему
6. Автоматический редирект → /profile ✅
7. Если попытается открыть / → Редирект в /profile
```

### Процесс 2: Существующий пользователь

```
1. Открывает / → Landing страница
2. Нажимает "Вход" → /auth/signin
3. Вводит email + пароль
4. Вход в систему → Редирект в /profile ✅
5. Если попытается открыть / → Редирект в /profile
```

### Процесс 3: Авторизованный пользователь

```
1. Уже авторизован
2. Открывает / → Автоматический редирект в /profile ✅
3. Работает в профиле
4. Выходит → Может вернуться на /
```

## 🛠️ Технические детали

### Код редиректа (app/page.tsx):

```typescript
export default function HomePage() {
  const { status } = useSession();

  // Редирект авторизованных пользователей в профиль
  useEffect(() => {
    if (status === 'authenticated') {
      window.location.href = '/profile';
    }
  }, [status]);

  // Показываем загрузку во время редиректа
  if (status === 'loading' || status === 'authenticated') {
    return (
      <div className="min-h-screen bg-[#0a1f1a] flex items-center justify-center">
        <div className="text-center">
          <div className="inline-block animate-spin rounded-full h-12 w-12 border-b-2 border-emerald-400 mb-4"></div>
          <div className="text-emerald-400 text-lg">
            {status === 'authenticated' ? 'Переход в профиль...' : 'Загрузка...'}
          </div>
        </div>
      </div>
    );
  }

  // Показываем Landing только для неавторизованных
  return <LandingPage />;
}
```

### Навигация (app/page.tsx):

```typescript
// Только кнопка "Вход" для неавторизованных

</Link>
```
