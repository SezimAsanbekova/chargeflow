# Быстрый старт - Управление автомобилями

## 🚗 Как добавить свой первый автомобиль

### Шаг 1: Перейдите в раздел автомобилей
1. Войдите в свой аккаунт
2. Откройте профиль
3. Нажмите на "Мои автомобили"

### Шаг 2: Добавьте автомобиль
1. Нажмите кнопку "Добавить" (зеленая кнопка справа вверху)
2. Заполните форму:

#### Основная информация:
- **Марка**: Tesla, Nissan, BYD, и т.д.
- **Модель**: Model 3, Leaf, Han, и т.д.
- **Год выпуска**: Выберите из списка

#### Характеристики зарядки:
- **Тип коннектора**: 
  - CCS2 - для большинства европейских авто
  - CHAdeMO - для Nissan Leaf, Mitsubishi
  - Type 2 - для AC зарядки
  - GB/T - для китайских авто
  
- **Максимальная мощность**: Сколько кВт может принять ваш автомобиль
  - Примеры: 50 кВт, 150 кВт, 250 кВт
  
- **Ёмкость батареи**: Полная ёмкость в кВт·ч
  - Примеры: 60 кВт·ч, 75 кВт·ч, 100 кВт·ч
  
- **Текущий заряд**: Передвиньте слайдер (0-100%)

3. Нажмите "Сохранить"

### Шаг 3: Готово! 🎉
Ваш первый автомобиль автоматически станет активным и будет использоваться для подбора зарядных станций.

## 🔄 Управление несколькими автомобилями

### Добавить второй автомобиль:
1. Откройте "Мои автомобили"
2. Нажмите "Добавить"
3. Заполните форму
4. Если хотите сделать его активным, поставьте галочку "Сделать активным автомобилем"

### Переключить активный автомобиль:
1. Откройте список автомобилей
2. Найдите нужный автомобиль
3. Нажмите "Сделать активным"

### Редактировать автомобиль:
1. Нажмите иконку карандаша на карточке автомобиля
2. Измените нужные поля
3. Нажмите "Сохранить изменения"

### Удалить автомобиль:
1. Нажмите иконку корзины на карточке
2. Подтвердите удаление

## ❓ Часто задаваемые вопросы

### Что такое "активный автомобиль"?
Активный автомобиль используется системой для:
- Подбора подходящих зарядных станций
- Фильтрации по типу коннектора
- Расчета времени зарядки
- Отображения в профиле

### Можно ли иметь несколько активных автомобилей?
Нет, только один автомобиль может быть активным. Вы можете легко переключаться между автомобилями.

### Что будет, если удалить активный автомобиль?
Система автоматически сделает активным первый оставшийся автомобиль из вашего списка.

### Где найти характеристики моего автомобиля?
- В руководстве пользователя
- На сайте производителя
- В приложении производителя
- На специализированных сайтах (ev-database.org)

### Можно ли изменить уровень заряда?
Да, вы можете в любой момент отредактировать автомобиль и изменить текущий уровень заряда.

## 🎯 Советы

1. **Точные характеристики**: Указывайте точные характеристики для корректного подбора станций
2. **Обновляйте заряд**: Периодически обновляйте текущий уровень заряда
3. **Несколько авто**: Если у вас несколько электромобилей, добавьте их все
4. **Правильный коннектор**: Убедитесь, что выбрали правильный тип коннектора
