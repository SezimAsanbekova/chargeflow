# Toast Уведомления при активации автомобиля

## ✅ Реализовано

Добавлены красивые всплывающие уведомления (Toast) при активации автомобиля.

---

## 🎯 Функциональность

### Когда показывается уведомление:

1. **При активации через кнопку "Сделать активным"** на странице списка автомобилей
2. **При активации через чекбокс** в форме редактирования автомобиля

### Что показывается:

```
✅ Активный автомобиль обновлён

Теперь при бронировании будет выбран CCS2 с мощностью 250 кВт

┌─────────────────────────────────────┐
│ 🚗 Tesla Model 3                    │
│                                     │
│ Коннектор: CCS2    Мощность: 250 кВт│
│ Батарея: 75 кВт·ч  Заряд: 80%      │
└─────────────────────────────────────┘

[OK]
```

---

## 🎨 Дизайн уведомления

### Расположение:
- **Позиция**: Правый нижний угол экрана
- **Анимация**: Плавное появление снизу вверх
- **Z-index**: 50 (поверх всех элементов)

### Стиль:
- **Фон**: `#0f2d26` (темно-зеленый)
- **Рамка**: 2px зеленая (`border-emerald-500`)
- **Тень**: Зеленое свечение (`shadow-emerald-500/20`)
- **Скругление**: 16px (`rounded-2xl`)
- **Максимальная ширина**: 448px (`max-w-md`)

### Элементы:

1. **Заголовок**:
   - Иконка галочки в зеленом круге
   - Текст: "✅ Активный автомобиль обновлён"
   - Кнопка закрытия (X)

2. **Описание**:
   - Информация о выбранном коннекторе и мощности
   - Цвет: светло-серый

3. **Карточка автомобиля** (только на странице списка):
   - Иконка автомобиля
   - Название: Марка + Модель
   - Сетка 2x2 с характеристиками:
     - Коннектор (зеленый)
     - Мощность (зеленый)
     - Батарея (белый)
     - Заряд (белый)

4. **Кнопка OK**:
   - Зеленая с прозрачностью
   - Закрывает уведомление

---

## ⏱️ Поведение

### Автоматическое закрытие:
- **На странице списка**: Закрывается через 5 секунд
- **На странице редактирования**: Закрывается через 2 секунды + переход к списку

### Ручное закрытие:
- Кнопка X в правом верхнем углу
- Кнопка OK внизу уведомления

### Анимация:
```css
@keyframes slide-up {
  from {
    transform: translateY(100%);
    opacity: 0;
  }
  to {
    transform: translateY(0);
    opacity: 1;
  }
}
```
- Длительность: 0.3 секунды
- Easing: ease-out

## 💻 Техническая реализация

### State управление:

```typescript
interface Toast {
  show: boolean;
  message: string;
  details: string;
  vehicle?: Vehicle;
}

const [toast, setToast] = useState<Toast>({ 
  show: false, 
  message: '', 
  details: '' 
});
```

### Показ уведомления:

```typescript
setToast({
  show: true,
  message: '✅ Активный автомобиль обновлён',
  details: `Теперь при бронировании будет выбран ${connectorType} с мощностью ${maxPowerKw} кВт`,
  vehicle: updatedVehicle,
});

// Автоматическое закрытие через 5 секунд
setTimeout(() => {
  setToast({ show: false, message: '', details: '' });
}, 5000);
```

### Закрытие уведомления:

```typescript
onClick={() => setToast({ show: false, message: '', details: '' })}
```

## 🎯 Сценарии использования

### Сценарий 1: Активация через список
1. Пользователь открывает список автомобилей
2. Нажимает "Сделать активным" на неактивном автомобиле
3. Появляется уведомление с полной информацией
4. Список обновляется (активный автомобиль выделен)
5. Уведомление автоматически закрывается через 5 секунд

### Сценарий 2: Активация через редактирование
1. Пользователь редактирует автомобиль
2. Ставит галочку "Активный автомобиль"
3. Нажимает "Сохранить изменения"
4. Появляется уведомление
5. Через 2 секунды переход к списку автомобилей

### Сценарий 3: Ручное закрытие
1. Уведомление появляется
2. Пользователь нажимает X или OK
3. Уведомление мгновенно закрывается
