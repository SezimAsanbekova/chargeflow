# Vehicle Management - Управление автомобилями

## ✅ Status: COMPLETED

Полноценная система управления автомобилями пользователя с возможностью добавления, редактирования, удаления и выбора активного автомобиля.

## 🎯 Реализованные функции

### 1. **Список автомобилей** (`/vehicles`)
- Отображение всех автомобилей пользователя
- Активный автомобиль выделен зеленой рамкой и меткой
- Карточки с полной информацией о каждом автомобиле
- Кнопки редактирования и удаления
- Кнопка "Сделать активным" для неактивных автомобилей
- Пустое состояние с призывом добавить первый автомобиль
- Информационный блок об активном автомобиле

### 2. **Добавление автомобиля** (`/vehicles/add`)
- Форма с валидацией всех полей
- **Основная информация**:
  - Марка (обязательно)
  - Модель (обязательно)
  - Год выпуска (выбор из списка)
- **Характеристики зарядки**:
  - Тип коннектора (CCS2, CHAdeMO, Type 2, GB/T)
  - Максимальная мощность зарядки (кВт)
  - Ёмкость батареи (кВт·ч)
  - Текущий уровень заряда (слайдер 0-100%)
- Чекбокс "Сделать активным автомобилем"
- Автоматическая активация первого автомобиля

### 3. **Редактирование автомобиля** (`/vehicles/edit/[id]`)
- Загрузка существующих данных
- Редактирование всех полей
- Изменение статуса активности
- Валидация при сохранении

### 4. **Удаление автомобиля**
- Подтверждение удаления
- Автоматическая активация другого автомобиля при удалении активного
- Каскадное удаление связанных данных (зарядные сессии)

### 5. **Управление активным автомобилем**
- Только один автомобиль может быть активным
- Кнопка "Сделать активным" на карточках
- Автоматическая деактивация предыдущего активного
- Визуальное выделение активного автомобиля

## 📊 Данные автомобиля

### Обязательные поля:
- ✅ Марка (brand)
- ✅ Модель (model)
- ✅ Год выпуска (year)
- ✅ Тип коннектора (connectorType)
- ✅ Максимальная мощность зарядки (maxPowerKw)
- ✅ Ёмкость батареи (batteryCapacityKwh)

### Опциональные поля:
- Текущий уровень заряда (currentChargeLevel) - по умолчанию 80%
- Активный статус (isActive) - автоматически для первого автомобиля

## 🔌 Типы коннекторов

1. **CCS2** (Combined Charging System) - Европейский стандарт
2. **CHAdeMO** - Японский стандарт (Nissan, Mitsubishi)
3. **Type 2** (Mennekes) - AC зарядка
4. **GB/T** - Китайский стандарт

## 🔄 API Endpoints

### GET `/api/vehicles`
Получить все автомобили пользователя
```json
{
  "vehicles": [
    {
      "id": "uuid",
      "brand": "Tesla",
      "model": "Model 3",
      "year": 2023,
      "connectorType": "CCS2",
      "maxPowerKw": 250,
      "batteryCapacityKwh": 75,
      "currentChargeLevel": 80,
      "isActive": true,
      "createdAt": "2024-01-01T00:00:00Z"
    }
  ]
}
```

### POST `/api/vehicles`
Добавить новый автомобиль
```json
{
  "brand": "Tesla",
  "model": "Model 3",
  "year": 2023,
  "connectorType": "CCS2",
  "maxPowerKw": 250,
  "batteryCapacityKwh": 75,
  "currentChargeLevel": 80,
  "isActive": false
}
```

### GET `/api/vehicles/[id]`
Получить один автомобиль

### PATCH `/api/vehicles/[id]`
Обновить автомобиль
```json
{
  "brand": "Tesla",
  "model": "Model 3 LR",
  "currentChargeLevel": 90,
  "isActive": true
}
```

### DELETE `/api/vehicles/[id]`
Удалить автомобиль

### POST `/api/vehicles/[id]/set-active`
Сделать автомобиль активным

## 📝 Бизнес-логика

### Активный автомобиль:
1. Первый добавленный автомобиль автоматически становится активным
2. Только один автомобиль может быть активным одновременно
3. При активации нового автомобиля, предыдущий деактивируется
4. При удалении активного автомобиля, активируется первый оставшийся

### Валидация:
- Мощность зарядки > 0 кВт
- Ёмкость батареи > 0 кВт·ч
- Уровень заряда: 0-100%
- Год выпуска: последние 30 лет
- Тип коннектора: только из списка разрешенных

## 🗄️ База данных

### Таблица: `vehicles`
```prisma
model Vehicle {
  id                 String        @id @default(uuid())
  userId             String
  brand              String
  model              String
  year               Int
  connectorType      ConnectorType
  maxPowerKw         Decimal
  batteryCapacityKwh Decimal
  currentChargeLevel Decimal
  isActive           Boolean       @default(true)
  createdAt          DateTime      @default(now())
  
  user               User          @relation(...)
  chargingSessions   ChargingSession[]
}
```

### Enum: `ConnectorType`
```prisma
enum ConnectorType {
  CCS2
  CHAdeMO
  Type2
  GB_T
}
```
