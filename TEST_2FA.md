# 🧪 Быстрый тест 2FA

## Запуск

```bash
npm run dev
```

Откройте: http://localhost:3000/auth/signin

---

## Тест 1: Регистрация с кодом ✅

### Шаг 1: Регистрация
1. Нажмите "Нет аккаунта? Зарегистрируйтесь"
2. Введите:
   - **Email:** `test2fa@example.com`
   - **Имя:** `Тест 2FA`
   - **Пароль:** `MySecure123!`
3. Нажмите "Зарегистрироваться"

### Шаг 2: Получение кода
**Ожидаемый результат:**
- Переход на страницу `/auth/verify-code`
- В терминале появится:
  ```
  ✅ Email успешно отправлен
  Subject: Код подтверждения регистрации ChargeFlow
  ```
- Скопируйте 6-значный код из HTML письма в терминале

### Шаг 3: Ввод кода
1. Введите код в 6 полей (или вставьте Ctrl+V)
2. Код отправится автоматически

### Шаг 4: Успешный вход
**Ожидаемый результат:**
- Переход в `/profile`
- Вы вошли в систему!

## Тест 2: Вход с кодом 🔐

### Шаг 1: Выход
1. Нажмите на ваше имя в правом верхнем углу
2. Нажмите "Выйти"

### Шаг 2: Вход
1. Откройте http://localhost:3000/auth/signin
2. Введите:
   - **Email:** `test2fa@example.com`
   - **Пароль:** `MySecure123!`
3. Нажмите "Войти"

### Шаг 3: Получение кода
**Ожидаемый результат:**
- Переход на страницу `/auth/verify-code`
- В терминале новый код

### Шаг 4: Ввод кода
1. Введите код
2. **Ожидаемый результат:** Вход в профиль

---

## Тест 3: Неправильный код ❌

### Шаг 1: Начните вход
1. Введите email и пароль
2. Перейдите на страницу кода

### Шаг 2: Введите неправильный код
1. Введите: `000000`
2. **Ожидаемый результат:**
   - Ошибка: "Неверный код"
   - Поля очищаются
   - Фокус на первом поле

## Тест 4: Повторная отправка 🔄

### Шаг 1: Начните вход
1. Введите email и пароль
2. Перейдите на страницу кода

### Шаг 2: Запросите новый код
1. Нажмите "Отправить повторно"
2. **Ожидаемый результат:**
   - Кнопка неактивна 60 секунд
   - В терминале новый код
   - Старый код больше не работает

### Шаг 3: Подождите
1. Подождите 60 секунд
2. **Ожидаемый результат:**
   - Кнопка снова активна
   - Можно запросить еще один код

---

## Тест 5: Вставка кода 📋

### Шаг 1: Начните вход
1. Введите email и пароль
2. Перейдите на страницу кода

### Шаг 2: Скопируйте код
1. Найдите код в терминале
2. Скопируйте его (например: `123456`)

### Шаг 3: Вставьте код
1. Кликните на любое поле
2. Нажмите Ctrl+V (или Cmd+V на Mac)
3. **Ожидаемый результат:**
   - Все 6 полей заполняются
   - Код отправляется автоматически
   - Вход в профиль

## Тест 6: Защита от брутфорса 🛡️

### Шаг 1: Создайте тестовый аккаунт
```
Email: bruteforce2fa@test.com
Пароль: TestPass123!
```

### Шаг 2: Попытки входа с неправильным паролем
1. Выйдите из аккаунта
2. Попробуйте войти с паролем: `WrongPass123!`
3. Повторите 5 раз

### Шаг 3: Блокировка
**Ожидаемый результат:**
- После 5-й попытки: "Слишком много неудачных попыток входа. Аккаунт заблокирован на 1 час"
- Код НЕ отправляется
- Страница кода НЕ открывается

## 🔍 Где смотреть код в терминале

После отправки кода в терминале будет:

```
✅ Email успешно отправлен: message-id
To: test2fa@example.com
Subject: Код подтверждения входа ChargeFlow
HTML: <!DOCTYPE html>
      <html>
        ...
        <div class="code">123456</div>  ← ВОТ ВАШ КОД!
        ...
      </html>
```

Найдите строку с `<div class="code">XXXXXX</div>` - это ваш код!

## 🐛 Проблемы?

### Код не приходит в терминал
**Решение:**
1. Проверьте, что сервер запущен (`npm run dev`)
2. Проверьте настройки EMAIL в `.env`
3. Если EMAIL не настроен, письма логируются в консоль

### Код не принимается
**Решение:**
1. Проверьте, что код правильный (6 цифр)
2. Проверьте, что код не истек (10 минут)
3. Проверьте, что код не был использован
4. Запросите новый код

### Не переходит на страницу кода
**Решение:**
1. Проверьте правильность email и пароля
2. Проверьте, что аккаунт не заблокирован
3. Посмотрите ошибки в консоли браузера (F12)
