## Как протестировать

### 1. Запустите приложение
```bash
npm run dev
```

### 2. Откройте http://localhost:3000

### 3. Проверьте кнопку "Войти"
- На главной странице в правом верхнем углу должна быть зеленая кнопка "Войти"
- Нажмите на нее - должна открыться страница `/auth/signin`

### 4. Тест регистрации
1. На странице входа переключитесь на "Нет аккаунта? Зарегистрируйтесь"
2. Заполните форму:
   - Имя: Тест
   - Email: test@example.com
   - Пароль: test123
3. Нажмите "Зарегистрироваться"
4. После успешной регистрации вы автоматически войдете и будете перенаправлены на главную

### 5. Проверьте UserMenu
После входа:
- В правом верхнем углу должно появиться меню пользователя
- Должен отображаться аватар/инициалы и баланс
- При клике открывается выпадающее меню с:
  - Информацией о пользователе
  - Балансом
  - Ссылками на Профиль и Бронирования
  - Кнопкой "Выйти"

### 6. Тест входа через Google (требует настройки)
1. Настройте Google OAuth (см. SETUP_AUTH.md)
2. Нажмите "Войти через Google"
3. Выберите Google аккаунт
4. После успешного входа вы будете перенаправлены на главную

### 7. Тест защищенных маршрутов
1. Попробуйте открыть `/profile` без авторизации
2. Вы должны быть перенаправлены на `/auth/signin?callbackUrl=/profile`
3. После входа вы вернетесь на `/profile`

### 8. Тест выхода
1. Нажмите на UserMenu
2. Нажмите "Выйти"
3. Вы должны быть перенаправлены на главную страницу
4. Кнопка "Войти" должна снова появиться

## Возможные проблемы

### Кнопка все еще не активна
- Очистите кеш браузера (Ctrl+Shift+R или Cmd+Shift+R)
- Перезапустите dev сервер

### Ошибка при регистрации
- Проверьте, что PostgreSQL запущен
- Проверьте DATABASE_URL в .env
- Выполните: `npm run db:migrate`

### UserMenu не появляется после входа
- Откройте DevTools (F12)
- Проверьте Console на ошибки
- Проверьте Network tab - должен быть запрос к `/api/auth/session`

### Google OAuth не работает
- Убедитесь, что GOOGLE_CLIENT_ID и GOOGLE_CLIENT_SECRET настроены в .env
- Проверьте redirect URI в Google Console
- Должен быть: `http://localhost:3000/api/auth/callback/google`

## Структура навигации

```
Главная (/)
├── Кнопка "Войти" (не авторизован)
└── UserMenu (авторизован)
    ├── Профиль (/profile)
    ├── Бронирования (/bookings)
    └── Выйти

Вход (/auth/signin)
├── Форма входа (email + пароль)
├── Вход через Google
└── Переключение на регистрацию

Регистрация (встроена в /auth/signin)
├── Форма регистрации
├── Регистрация через Google
└── Переключение на вход

Профиль (/profile) - защищен
├── Информация о пользователе
├── Баланс
└── Быстрые действия
```

## Cookies

После успешного входа в браузере должны быть установлены cookies:
- `next-auth.session-token` (httpOnly, secure в production)
- `next-auth.csrf-token`

Проверить можно в DevTools → Application → Cookies → http://localhost:3000
