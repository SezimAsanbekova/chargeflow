# ✅ Админ-панель ChargeFlow - Готово!

## 🎉 Что реализовано

Полноценная админ-панель с профессиональной двухфакторной аутентификацией через Telegram.

## 📋 Быстрая настройка

### 1. Telegram бот
```
@BotFather → /newbot → Сохраните токен
```

### 2. Telegram ID
```
@userinfobot → Сохраните ID
```

### 3. База данных
```bash
# Отредактируйте prisma/seed-admin.ts
npm run db:seed:admin
```

### 4. .env
```env
JWT_SECRET="ваш_секретный_ключ"
```

### 5. Диалог с ботом
```
Найдите бота → /start
```

### 6. Вход
```
http://localhost:3000/admin/signin
```

## 🔒 Безопасность

### Процесс входа
```
1. Email + Telegram ID
   ↓
2. Проверка в БД (роль = admin)
   ↓
3. Генерация 6-значного кода
   ↓
4. Отправка через Telegram Bot API
   ↓
5. Ввод кода
   ↓
6. Проверка кода (одноразовый, 10 минут)
   ↓
7. Генерация JWT токена
   ↓
8. Установка httpOnly cookie
   ↓
9. Вход в админ-панель ✅
```

### Защита
- Админы не могут зайти на `/profile`, `/vehicles`, `/booking`, `/charging`
- Пользователи не могут зайти на `/admin/*`
- Автоматические перенаправления
- Проверка токенов на каждом запросе
